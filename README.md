# AquaControl Plugin para FacturaScripts

Plugin para gestión de suministros de agua, medidores y lecturas.

## Características
- Gestión de titulares/clientes
- Administración de suministros de agua
- Control de medidores
- Registro de lecturas y consumo

## Requisitos
- FacturaScripts 2025 o superior
- PHP 7.4 o superior

## Instalación
1. Descargar el plugin
2. Copiar la carpeta `AquaControl` a `Plugins/` de FacturaScripts
3. Activar desde el panel de control de FacturaScripts

## Tablas de base de datos
El plugin crea las siguientes tablas:
- `ac_titulares` - Titulares de suministros
- `ac_suministros` - Suministros de agua
- `ac_medidores` - Medidores instalados
- `ac_lecturas` - Registro de lecturas
